# J.A.R.V.I.S. Voice Assistant

Голосовой ассистент в стиле JARVIS из Iron Man для Android.

## Возможности

- **Голосовое управление** — говорите команды, JARVIS выполняет
- **AI интеграция** — подключение к ChatGPT для умных ответов
- **Голографический интерфейс** — анимированное кольцо как в фильме
- **Управление телефоном** — звонки, SMS, будильники, приложения

## Команды

| Команда | Описание |
|---------|----------|
| "Позвони маме" | Совершить звонок |
| "Отправь SMS папе Привет!" | Отправить сообщение |
| "Поставь будильник на 7:00" | Установить будильник |
| "Открой камеру" | Открыть приложение |
| "Который час?" | Узнать время |
| "Какая погода?" | Узнать погоду |
| "Помощь" | Список всех команд |

## Установка

1. Откройте проект в Android Studio
2. Дождитесь синхронизации Gradle
3. Подключите Android устройство
4. Нажмите Run ▶

## Настройка AI

1. Получите API ключ на platform.openai.com
2. В приложении нажмите "⚙ Настройки AI"
3. Введите ключ
4. Теперь JARVIS отвечает на любые вопросы!

## Требования

- Android 8.0+ (API 26)
- Microphone permission
- Internet (для AI)

## Архитектура

- **VoiceEngine** — распознавание речи и синтез
- **CommandParser** — парсинг команд из текста
- **CommandHandler** — выполнение команд на устройстве
- **AiBackend** — интеграция с OpenAI API
- **Room DB** — хранение истории сообщений
